Amytis provides beautiful syntax highlighting for dozens of programming languages.

## TypeScript

```typescript
interface Post {
  title: string;
  date: string;
  featured: boolean;
}

function getFeatured(posts: Post[]): Post[] {
  return posts.filter(p => p.featured);
}
```
